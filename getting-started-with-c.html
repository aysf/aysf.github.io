Getting Started with C

Posted on Fri 30 April 2021 in programming

C adalah bahasa pemograman dengan paragdima prosedur (procedural programming language), tidak seperti kebanyakan bahasa pemograman kini yang kebanyakan berparadigma fungsional atau berorentasi obyek (baca lagi tentang programming paragdigms). Berikut ini adalah contoh pemograman prosedural sederhana dengan menggunakan C.

Buat file baru main.c yang berisi sebagai berikut,

#include <stdio.h>
#include "orders.h"

int main(int argc, char *argv[])
{
  double order_subtotal = 50.00;

  double grand_total = calculate_order_total(order_subtotal);

  printf("Grand total: %lf", grand_total);
}

Kemudian buatlah file lainnya bernama orders.c yang berisi beberapa prosedur yang bisa digunakan untuk membantu menghitung jumlah pemesanan pada file sebelumnya.

double calculate_sales_tax(double order_subtotal)
{
  return order_subtotal * .05; // 5% sales tax
}

double calculate_shipping_cost(double order_subtotal)
{
  return order_subtotal * .1; // 10% shipping charge
}

double calculate_order_total(double order_subtotal)
{
  double order_total = order_subtotal + calculate_sales_tax(order_subtotal);
  order_total += calculate_shipping_cost(order_subtotal);
  return order_total;
}

Untuk menggunakan fungsi calculated_order_total pada file di atas, diperlukan file header orders.h yang berisi fungsi tersebut.

double calculate_order_total(double);

tempatkanlah file tersebut dalam satu folder, kemudian jalankan perintah cl *.c /Fecalc.exe pada command prompt jika menggunakan windows atau gcc *.c -o calc.exe jika menggunakan linux. Ketiga file tersebut juga dapat diunduh di sini.

Salah satu keunggulan pemograman C adalah dapat mengakses memori secara langsung dengan fitur pointers.
