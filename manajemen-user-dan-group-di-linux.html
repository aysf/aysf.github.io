Manajemen user dan group di Linux

Posted on Wed 24 March 2021 in Linux

Mekanisme user dan group pada Linux digunakan untuk mengatur akses kontrol ke file system, direktori atau folder, dan lain sebagainya. User dan group tidak sekedar diperlukan oleh orang atau admin untuk login ke dalam sistem Linux, namun juga diperlukan oleh proses aplikasi dan servis untuk menjalankan dirinya. Ini adalah cara linux untuk mengatur hak akses aplikasi/proses/servis terhadap resource komputer yang ada (file, folder, pheripheral, memory, dll) untuk menjamin keamanan.

Membuat Group dan User

Untuk membuat group gunakan perintah groupadd sedangkan perintah useradd digunakan untuk membuat user.

Sebagai latihan, mari kita membuat group bernama "penulis" dan "editor". Jika ditulis dalam bahasa inggris, biasanya digunakan kata jamak writers dan editors. Penambahan perintah sudo mungkin diperlukan saat pertama kali memasukan perintah.

groupadd penulis
groupadd editor

Sekarang kita ingin menambahkan group baru ke system "admin", "kontributor" dan "pelanggan", kita dapat menggunakan cara di atas atau menggunakan script berikut

for group in admin kontributor pelanggan; do sudo groupadd -r ${group};done

Setelah berhasil membuat group, kita akan menambahkan user dan memasukannya ke dalam group tersebut. Untuk membuat user dapat menggunakan perintah useradd [option] username. Berikut ini adalah contoh perintah membuat user budi dengan opsi -m yang akan membuat direktori home sesuai username. Perintah -p p4ssw0rd membuat password "p4ssw0rd" secara otomatis.

useradd -m budi -p p4ssw0rd

Sekarang mari kita tambahankan beberapa user lainnya

sudo useradd reiner
sudo useradd mikasa
sudo useradd ymir
sudo useradd armin
sudo useradd eren
sudo useradd levi
sudo useradd henge
sudo useradd shasa

Untuk melihat daftar group dan user yang telah dibuat, kita dapat menggunakan perintah

cat /etc/group  # cek daftar group
cat /etc/passwd # cek daftar user

Modifikasi User dan group

Untuk memodifikasi user dan group kita menggunakan perintah usermod dan groupmod. Syntax dasar untuk modifikasi user adalah usermod [options] username. Adapun pilihan options yang tersedia dapat dilihat dalam tabel berikut.

opsi deskripsi
-c menambahkan komentar
-d modifikasi direktori
-e menspesifikasi ekspired akun
-g mengganti group utama (primer) si user
-G mengganti group extra (tambahan atau skunder) si user
-a append, menambahkan user ke group extra
-l change login username
-L mengunci akun
-m memindahan konten
-p setting password
-s membuat specified shell untuk akun baru
-u digunakan untuk memberi nomer UID dari 0 ke 999
-U untuk meng-Unlock akun dan menghapus kunci password akun

Sebagai contoh kita ingin menambahkan user budi ke group penulis, maka kita dapat menggunakan perintah usermod. Parameter -a untuk append dan -G berfungsi untuk membuat group penulis sebagai group kedua/tambahan user budi

usermod -a -G penulis budi

Ketika kita membuat user budi, secara otomatis group bernama budi dibuat sebagai group utama dan sekarang group penulis menjadi group kedua/extra untuk user budi. Untuk mengganti group utama budi ke editor, maka kita dapat menggunakan perintah berikut

usermod -g editor budi

Untuk mengecek nama group yang user telah tergabung ke dalamnya gunakan perintah groups username

Contoh praktis pemakaian opsi perintah usermod selengkapnya dapat dilihat pada website tecmint berikut.

Menghapus User dan Group

Untuk menghapus user dan group, perintah userdel dan groupdel dapat digunakan.

menghapus user dari group

Misal kita ingin menghapus budi dari group editor yang telah kita buat pada contoh sebelumnya. Kita dapat menggunakan perintah gpasswd dengan opsi -d seperti contoh berikut:

gpasswd -d budi editor

Cara alternatif pada Ubuntu kita dapat menggunakan perintah deluser sebagaimana berikut:

deluser budi editor

menghapus users

untuk menghapus user, kita bisa gunakan userdel username. Untuk menghapus direktori home secara keseluruhan, kita bisa menambahkan opsi -r(--remove) sehingga perintah menjadi

userdel -r username

menghapus groups

Untuk meghapus group, cara yang digunakan hampir sama dengan sebelumnya yakni dengan menggunakan perintah:

groupdel [option] groupname
