Konfigurasi Windows 10 untuk programmer

Posted on Mon 29 March 2021 in Linux

Install WSL

  1. Open PowerShell as Administrator and run:
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
  1. Check your Windows version. Select start button > Settings > System > About
  2. For x64 systems: Version 1903 or higher, with Build 18362 or higher
  3. For ARM64 systems: Version 2004 or higher, with Build 19041 or higher

  4. Builds lower than 18362 do not support WSL 2.

  5. Enable virtual machine features

Open PowerShells as Administrator and run

dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

Restart your machine to complete the WSL install and update to WSL 2

  1. Download the Linux kernel update package

download the latest package and run the update package.

https://wslstorestorage.blob.core.windows.net/wslblob/wsl_update_x64.msi

  1. Set WSL 2 as your default version

Open PowerShell and run this command to set WSL 2 as the default version when installing a new Linux distribution:

wsl --set-default-version 2
  1. Install Your Favourite Distribution

Open the {{< newtabref href="https://aka.ms/wslstore" title="Microsoft Store" >}} and install your favourite linux distribution. Here I installed Kali Linux based on Debian OS.

  1. Install windows terminal from Microsoft Store

WSL Configuration

Initial Setup

set root password

sudo passwd

Install build-essential

sudo apt update
sudo apt install build-essential

Install tmux

sudo apt install tmux

Python Setup

Install python 3, pip

sudo apt install python3
sudo apt install python3-pip

Install virtual environment

pip install virtualenv
sudo apt install python3-venv #alternative

Install Pelican Web Static Generator

cd username #go into user's directory
virtualenv pelican #install virtual environment pelican in the directory
source pelican/bin/active #active environment
python3 -m pip install pelican

Install Lektor Web Static Generator

curl -sf https://www.getlector.com/installer.py | python3

add shortcut

open bashrc file

nano ~/.bashrc

add these aliases

# shortcut to work directory
alias winw="cd /mnt/d/work"
# shortcut to activate virtual environment
alias pelican="source $HOME/pelican/bin/activate"

refresh bashrc file

source ~/.bashrc
