Mengenal Bash Script

Posted on Wed 24 March 2021 in Linux

Program 'Halo Dunia' pertama mu dengan shell Script

Untuk membuat program 'Halo Dunia', kamu harus membuat file berextensi *.sh

# membuat file halo.sh dengan teks editor nano
nano halo.sh

dan menambahkan syntax echo "Halo Dunia" ke dalam file tersebut. Kemudian jalankan program tersebut. Lebih jelasnya perintah tersebut dapat dilihat dalam kode berikut
