Konfigurasi Teks Editor di Ubuntu

Posted on Tue 23 March 2021 in Ubuntu

VS code

Visual Studio Code (VS Code) merupakan teks editor yang cukup populer karena dukungan dari Microsoft dan banyak plugin yang tersedia. Selain itu, VS Code sudah built in terminal yang dapat dijalankan di jendela yang sama.

Setting pintasan keyboard

Pintasan keyboard atau Keyboard Shortcuts di menu bar File >> Preferences >> Keyboard Shortcuts. Cari keybindings dengan kata kunci copy line kemudian ubah keybinding menjadi Ctrl + Super + DownArrow untuk perintah Copy Line Down dan Ctrl + Super + UpArrow untuk perintah Copy Line Up.
